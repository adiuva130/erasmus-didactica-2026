# ErasmusDidactica 2026

**Erasmus+ KA1 — Teacher Training Mobility Project**

## Project Structure

```
├── docs/
│   ├── application/      # KA1 application forms, project description
│   ├── reports/           # Interim and final reports
│   └── dissemination/     # Dissemination plan, publications, outreach
├── src/
│   ├── curriculum/        # Training curriculum and learning outcomes
│   ├── materials/         # Teaching materials, slides, handouts
│   └── assessments/       # Assessment tools, evaluation forms
├── admin/
│   ├── budget/            # Budget plans, financial documents
│   ├── agreements/        # Inter-institutional agreements, grants
│   └── certificates/      # Europass, certificates of attendance
└── meetings/              # Meeting notes, agendas, minutes
```

## About

ErasmusDidactica is a KA1 teacher training mobility project under the Erasmus+ programme, focused on professional development and international collaboration for educators.
